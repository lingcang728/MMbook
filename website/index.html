关于阅读这件事

真正的阅读不是把眼睛贴在屏幕上，
而是把灵魂放在文字里。

当你滑过这一行，世界为它静下来 ——
这是 MMbook 独创的聚光灯模式，让此刻读到的段落自然亮起，无关的内容温柔淡出。

于是你不再走神，
你只是，
在读。

const read = (book) => book.absorb();

为阅读而做的每一处雕琢

不是又一个编辑器，是一个真正只为"读"服务的工具。

👁️

聚光灯专注模式

当前段落 100% 高亮，前后段落 40% / 15% 三层渐变淡出，配合 300ms 缓动过渡。读到哪，光在哪。

🎨

三对精心调色的主题

素纸、墨石、暮光 —— 每对主题都有深浅两色，色彩与字距经过反复调校，每个字都站得稳。

🔍

悬浮式全局搜索

macOS 风格的圆角搜索框，实时高亮命中位置，一键跳转。Ctrl+F 即可呼出。

📑

智能目录大纲

自动提取 Markdown 标题层级，侧边栏目录支持一键跳转，再长的文档也不会迷路。

✍️

双击即编辑

读着读着发现错别字？双击段落直接进入编辑态，改完即存。轻量、克制、随时回到阅读流。

⚡

Rust + Tauri 极致性能

毫秒级冷启动，几十兆内存占用，安装包 < 10MB。比浏览器轻一个数量级。

主题不止是配色，是氛围

三对主题，一套阅读哲学。每一对都同时调好深浅色，让眼睛在白天与深夜都得到善待。
